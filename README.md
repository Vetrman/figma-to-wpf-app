# Figma -> Windows приложение

Проект по вашему ТЗ готов. Вот что получилось:

---

## Что сделано по ТЗ

1. **Дизайн точь-в-точь** - Как в Figma макете  
2. **Только внешний вид** - Никакой логики, только верстка  
3. **Под Windows** - WPF приложение на .NET 8.0  
4. **Все элементы на месте** - Frame, Button, Welcome Text, Play Icon  
5. **Цвета правильные** - #0D99FF, #9747FF, #34C759, #6099FF  
6. **Без лишних библиотек** - Только стандартный WPF  
7. **Размер окна 1000×470** - Как в макете  

---

## Техническое

- **Окно:** 1000×470 пикселей  
- **Фон:** #0D99FF (Primary Blue)  
- **Кнопка:** "Get Started", меняет цвет при клике  
- **Шрифт:** Inter 16px (или Segoe UI если Inter нет)  
- **Статус бар:** "Ready • 1000 × 470 • @1x"  

---

## Как запустить

**Если у вас Visual Studio:**
1. Скачайте архив с проектом
2. Откройте `Figma.sln`
3. Нажмите F5

**Если через консоль:**
```bash
dotnet build
dotnet run
